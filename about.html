About EnglishHub

Empowering Students Through English Learning

Tentang Project

EnglishHub adalah platform blog pembelajaran Bahasa Inggris yang dikembangkan sebagai project tugas mata kuliah Bahasa Inggris, Semester 5, Telkom University. Project ini bertujuan untuk menyediakan konten edukatif berkualitas yang membantu mahasiswa meningkatkan kemampuan Bahasa Inggris mereka.

Website ini dibangun menggunakan teknologi frontend modern (HTML5, CSS3, JavaScript) dengan pendekatan static/client-side only, sehingga dapat diakses dengan mudah tanpa memerlukan backend atau database.

Konten yang Kami Tawarkan

EnglishHub menyediakan berbagai artikel edukatif yang mencakup:

  • Grammar: Penjelasan lengkap tentang tata bahasa Inggris dengan contoh praktis
  • Vocabulary: Strategi efektif untuk memperluas kosakata Bahasa Inggris
  • Pronunciation: Tips dan teknik untuk meningkatkan pengucapan yang jelas
  • Academic Writing: Panduan menulis akademik untuk keperluan kuliah dan penelitian
  • Technology & English: Artikel tentang penggunaan Bahasa Inggris dalam dunia teknologi

Informasi Teknis

Website ini dikembangkan dengan spesifikasi berikut:

  • Teknologi: HTML5, CSS3, JavaScript (Vanilla)
  • Tipe: Static Frontend-Only Website
  • Data: Client-side JSON (No Backend/Database)
  • Desain: Modern, Responsive, dan User-Friendly
  • Deployment: Dapat di-host di Vercel, Netlify, atau GitHub Pages
